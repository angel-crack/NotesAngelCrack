[+] Crar proyecto de Node:

```powershell
npm init --yes
```

[+] Instalamos Express:

```powershell
npm install express --save
```

[+] Crea la siguiente estructura:

```powershell
00/  
┣ node_modules/  
┣ package-lock.json  
┗ package.json
```

Y creamos un archivos .js

```js
import express from 'express';
const app = express();
```

```powershell
00/  
┣ node_modules/  
┣ app.js  
┣ package-lock.json  
┗ package.json
```
